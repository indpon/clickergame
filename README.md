# Clicker Game

A simple clicker game built with raylib as a way to learn C++ and raylib. This is a personal learning project and is not meant to be an example of best practices.

## About

This project was made to get hands-on experience with:
- C++ basics
- raylib and raylib-cpp
- raygui for UI elements
- CMake as a build system

## Dependencies

- [raylib](https://github.com/raysan5/raylib)
- [raylib-cpp](https://github.com/RobLoach/raylib-cpp)
- [raygui](https://github.com/raysan5/raygui)
- CMake 3.14+
- A C++ compiler (gcc, clang, or MSVC)

## Building

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

Then run the executable:

```bash
./game
```

## Gameplay

- Click the button to earn clicks
- Spend clicks on upgrades to earn more clicks per click
